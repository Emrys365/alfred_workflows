# alfred_workflows

### 1. Simplify fractions

#### Usage: 

+ Convert a given decimal to a simplified fraction
+ Simplify a given fraction

#### Examples:

![demo](Simplify_fractions/demo.png)

+ `frac .11`  ==>  `.11 = 11 / 100`
+ `frac 4/6`  ==>  `4/6 = 2 / 3`
+ `frac -1.4/2.2`  ==>  `-1.4/2.2 = -7 / 11`
